## Suggested Installation

Clone the repository:
```bash
$ git clone <repository_URL> emicore 
```
Move into the new repository and create a new environment:
```bash
$ cd emicore
$ python -m venv venv 
```

Installing the module with pip:
```bash
$  ./venv/bin/pip3 install -e .   
```
